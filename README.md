# twitter-spelling-nazi
just a little sideproject

In order to run it, you need to create a twitter.json file in secrets directory.
```
{
  "consumer_key": "...",
  "consumer_secret": "...",
  "access_token": "...",
  "access_token_secret": "..."
}
```

Then just 
```
npm i
jspm i
npm start
```

Tested only in latest Chrome version 41.0.2272.101 m.
